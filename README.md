# secure_windows
# Summary
The list shown below contains all of the NIST 800 Revision 4 controls that are covered by hardening Windows 2012 and 2016 Standalone, Member, and Domain Control servers.
## Controls
Each control is ranked as either Low, Moderate, or High impact, based on the potential damage severity if the control is not implemented. Certain controls and control attributes are not required by the National Institute of Standards and Technology (NIST), but are covered to provide additional security for participating systems. Both Autostructure and Microsoft Azure covers many of the NIST controls.
